# Tmall-Order-Assistant
天猫下单助手-Chrome插件
为了方便和简化抢票业务逻辑，提高代码的效率，此插件只负责提交抢票和发起付款请求。

# 抢票方法
1. 通过手机端将天猫超市端的茅台加入购物车，等待插件自动运行。
2. 插件自动运行后，时间到了后，看看是否订单里面有付款项，有的话就说抢票成功，没有则是没中。
